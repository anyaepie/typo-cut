How to Use

Getting started with Typocut is easy:

  1. Type your text in the input field
  2. Enjoy seamless line breaks for your text
  3. Watch out for maximum possible text lenght at these letter sizes
  4. Adjust letter size, spacing, and noise using GUI controls
  5. Choose between gradient or uploaded images
  6. Upload images to choose between gradients or uploaded images
  7. Upload up to 15 images JPG, JPEG, PNG each less than 2MB
  8. Reset images by re-uploading again
  9. Check-out inverted mask to get reverse cutouts
  10. Drag letters to reposition them and save as PNG
  11. Generate, preview and save A4 alphabet stickers
  12. A4 alphabet uses English alphabet letter frequencies

Experiment with different settings to have fun!

About

Typocut mixes collage, typography and randomness.

Each letter is a canvas, randomly cropped from configurable gradients or uploaded images, with adjustable noise and style parameters.

Letters are built using strict 3x3 grid with primitives (rectange, triangle, cut-out rectangles, half-circles) that are flipped, rotated - all with noise applied, combining pixel and parametric fonts.

This tool built out of my personal liking to use cut-out letters in my collages.
